2

Download as ZIP

Required ~25 min

Export all generated files as a downloadable ZIP archive.

Implementation Guide

File to modify: src/components/HeaderActions.tsx
Install package: npm install jszip
Imports needed: 
import JSZip from "jszip"
import { Download } from "lucide-react"
import { useFileSystem } from "@/lib/contexts/file-system-context"
Key code: 
const { getAllFiles } = useFileSystem()

async function handleDownload() {
  const files = getAllFiles()  // Map<string, string>
  const zip = new JSZip()

  files.forEach((content, path) => {
    zip.file(path.slice(1), content)  // Remove leading "/"
  })

  const blob = await zip.generateAsync({ type: "blob" })
  const url = URL.createObjectURL(blob)
  const a = document.createElement("a")
  a.href = url
  a.download = "uigen-export.zip"
  a.click()
}

Checklist

  • ☐ JSZip installed
  • ☐ Download icon button in header
  • ☐ getAllFiles() retrieves files
  • ☐ ZIP downloads automatically

Note

getAllFiles() returns Map<string, string>

Keys are paths like "/src/App.tsx"

Use path.slice(1) to remove leading slash

Keyboard Shortcuts

Bonus ~25 min

Add Cmd/Ctrl+K command palette for quick actions.

Implementation Guide

Create new file: src/components/CommandPalette.tsx
Or modify: src/app/page.tsx
Imports needed: 
import { useEffect, useState } from "react"
import { CommandDialog, CommandInput, CommandList,
         CommandEmpty, CommandGroup, CommandItem } from "@/components/ui/command"
Key code: 
const [open, setOpen] = useState(false)

useEffect(() => {
  const handler = (e: KeyboardEvent) => {
    if ((e.metaKey || e.ctrlKey) && e.key === "k") {
      e.preventDefault()
      setOpen(true)
    }
  }
  window.addEventListener("keydown", handler)
  return () => window.removeEventListener("keydown", handler)
}, [])

// CommandDialog with CommandGroup containing:
// - CommandItem onSelect={() => handleClear()}>Clear All
// - CommandItem onSelect={() => handleDownload()}>Download ZIP

## UIGen Project Learnings

### Context Patterns
- FileSystemProvider must wrap components using useFileSystem()
- reset() clears files + selectedFile + triggers refresh

### Dialog Component
- Uses Radix UI, import from @/components/ui/dialog
- DialogContent contains all modal content

### File System API
- getAllFiles() returns Map<string, string>
- Paths have leading slash, remove with path.slice(1)

### Keyboard Events in React
- Always cleanup listeners in useEffect return
- Check e.metaKey (Mac) OR e.ctrlKey (Windows)
