FEATURE STAGES

1

Clear All Files

Required ~20 min

Add a "Clear All" button with confirmation dialog to reset all generated files.

Requirements

  • ✓ Add "Clear All" button with Trash2 icon to header
  • ✓ Show confirmation dialog: "Are you sure?"
  • ✓ "Delete All" button calls reset()
  • ✓ "Cancel" button closes dialog without action

Expected Behavior

1. User clicks "Clear All" button

2. Warning dialog appears (red theme)

3. "Delete All" → All files removed, tree reset

4. "Cancel" → Dialog closes, no changes

2

Download as ZIP

Required ~25 min

Export all generated files as a downloadable ZIP archive.

Requirements

  • ✓ Add "Download ZIP" button with Download icon
  • ✓ Get all files via getAllFiles()
  • ✓ Create ZIP (JSZip or Blob API)
  • ✓ Filename: uigen-export.zip

Expected Behavior

1. User clicks "Download ZIP" button

2. getAllFiles() returns Map<path, content>

3. ZIP file created with all files

4. Browser auto-downloads the ZIP

3

Keyboard Shortcuts

Bonus ~25 min

Add Cmd/Ctrl+K command palette for quick actions.

Requirements

  • ✓ Cmd/Ctrl+K opens Command Palette
  • ✓ Use existing Command component
  • ✓ Include: "Clear All Files" command
  • ✓ Include: "Download as ZIP" command

Expected Behavior

1. Press Cmd+K (Mac) or Ctrl+K (Win)

2. Command Palette modal opens

3. Search filters available commands

4. Select command → executes & closes

CLAUDE.MD EXPECTATIONS

Your CLAUDE.md should document learnings from errors and discoveries. Good examples:

Example CLAUDE.md Entries
## UIGen Project Learnings

### Context Patterns
- FileSystemProvider must wrap components using useFileSystem()
- reset() clears files + selectedFile + triggers refresh

### Dialog Component
- Uses Radix UI, import from @/components/ui/dialog
- DialogContent contains all modal content

### File System API
- getAllFiles() returns Map<string, string>
- Paths have leading slash, remove with path.slice(1)

### Keyboard Events in React
- Always cleanup listeners in useEffect return
- Check e.metaKey (Mac) OR e.ctrlKey (Windows)
